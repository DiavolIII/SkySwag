# User Stories для SkySwag

## Роли пользователей

| Роль | Описание |
|------|----------|
| **Гость** | Неавторизованный посетитель сайта |
| **Клиент** | Зарегистрированный пользователь |
| **Менеджер** | Сотрудник компании SkySwag |
| **Администратор** | Управляет системой и пользователями |

---

## User Stories

### Гость

| ID | User Story | Приоритет |
|----|------------|-----------|
| US-001 | Как гость, я хочу просматривать каталог вертолетов, чтобы ознакомиться с ассортиментом | Высокий |
| US-002 | Как гость, я хочу фильтровать вертолеты по цене, чтобы найти варианты в своем бюджете | Высокий |
| US-003 | Как гость, я хочу просматривать 3D модели вертолетов, чтобы оценить их внешний вид | Средний |
| US-004 | Как гость, я хочу отправлять заявку на оценку вертолета, чтобы узнать его стоимость | Высокий |
| US-005 | Как гость, я хочу читать новости компании, чтобы быть в курсе событий | Низкий |

### Клиент

| ID | User Story | Приоритет |
|----|------------|-----------|
| US-006 | Как клиент, я хочу зарегистрироваться, чтобы сохранять историю заказов | Средний |
| US-007 | Как клиент, я хочу оформлять заказ на вертолет, чтобы приобрести его | Высокий |
| US-008 | Как клиент, я хочу просматривать свои заказы, чтобы отслеживать их статус | Средний |
| US-009 | Как клиент, я хочу сохранять понравившиеся модели в избранное | Низкий |
| US-010 | Как клиент, я хочу получать уведомления о изменении статуса заказа | Низкий |

### Менеджер

| ID | User Story | Приоритет |
|----|------------|-----------|
| US-011 | Как менеджер, я хочу просматривать заявки клиентов, чтобы обрабатывать их | Высокий |
| US-012 | Как менеджер, я хочу изменять статус заказа, чтобы информировать клиента | Высокий |
| US-013 | Как менеджер, я хочу добавлять новые вертолеты в каталог | Средний |
| US-014 | Как менеджер, я хочу редактировать информацию о вертолетах | Средний |

### Администратор

| ID | User Story | Приоритет |
|----|------------|-----------|
| US-015 | Как администратор, я хочу управлять пользователями (блокировать/разблокировать) | Высокий |
| US-016 | Как администратор, я хочу просматривать статистику продаж | Средний |
| US-017 | Как администратор, я хочу управлять ролями пользователей | Высокий |
| US-018 | Как администратор, я хочу просматривать логи системы | Низкий |
