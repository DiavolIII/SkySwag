## ✨ О проекте

**SkySwag** - это современная веб-платформа для премиальной продажи вертолетов с интерактивной 3D визуализацией. Проект разработан в рамках курсовой работы и демонстрирует интеграцию передовых веб-технологий для создания уникального пользовательского опыта.

## 📦 Установка и запуск

### Предварительные требования

| Требование | Версия | Ссылка |
|------------|--------|--------|
| Python | 3.12+ | [python.org](https://python.org) |
| Git | 2.40+ | [git-scm.com](https://git-scm.com) |
| Браузер | Современный | Chrome/Firefox/Edge |

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/DiavolIII/SkySwag.git
cd SkySwag

# Запуск сервера (Python)
python server.py

# Открыть в браузере
start http://localhost:8000/catalog.html

# Просто запустите
start-python.bat
```
### Запуск тестов
cd backend
python -m unittest tests/test_api.py -v
